# Как выложить приложение онлайн

Эта инструкция для проекта `board-games-online`.

## 1. GitHub

1. Открой https://github.com.
2. Создай аккаунт или войди.
3. Нажми `New repository`.
4. Назови репозиторий, например `board-games-online`.
5. Сделай его `Public` или `Private`.
6. Нажми `Create repository`.

Позже сюда нужно будет загрузить папку `board-games-online`.

## 2. Supabase

1. Открой https://supabase.com.
2. Создай аккаунт или войди.
3. Нажми `New project`.
4. Выбери организацию.
5. Назови проект, например `board-games`.
6. Придумай пароль базы данных и сохрани его у себя.
7. Выбери ближайший регион.
8. Нажми `Create new project`.

## 3. Создать таблицы

1. В Supabase открой свой проект.
2. Слева нажми `SQL Editor`.
3. Нажми `New query`.
4. Открой файл `supabase/schema.sql` из этой папки.
5. Скопируй весь текст.
6. Вставь его в Supabase SQL Editor.
7. Нажми `Run`.

После этого появятся таблицы, роли, хранилище обложек и защита от повторной оценки.

## 4. Включить себя как владельца

1. Когда сайт будет запущен, войди в него своим e-mail.
2. Вернись в Supabase.
3. Открой `SQL Editor`.
4. Выполни команду, заменив e-mail на свой:

```sql
update public.profiles set role = 'owner' where email = 'your-email@example.com';
```

После этого ты сможешь добавлять и редактировать игры.

## 5. Найти ключи Supabase

1. В Supabase открой `Project Settings`.
2. Открой `API`.
3. Скопируй:
   - `Project URL`;
   - `anon public` key.

Это не пароль администратора. Это публичные данные для сайта.

## 6. Vercel

1. Открой https://vercel.com.
2. Войди через GitHub.
3. Нажми `Add New` -> `Project`.
4. Выбери репозиторий `board-games-online`.
5. В разделе `Environment Variables` добавь:

```text
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
```

6. Вставь туда значения из Supabase.
7. Нажми `Deploy`.

После этого Vercel даст ссылку на сайт.

## 7. Как будут заходить друзья

1. Друг открывает ссылку сайта.
2. Вводит свой e-mail.
3. Получает письмо со ссылкой для входа.
4. Заходит и ставит оценку.
5. Повторно оценить ту же игру с этого аккаунта он уже не сможет.

## 8. Что дальше

Когда сайт будет опубликован, можно улучшать:

- красивую страницу приглашения для друзей;
- список друзей;
- удаление или скрытие оценок;
- несколько коллекций игр;
- права доступа только по приглашению.
